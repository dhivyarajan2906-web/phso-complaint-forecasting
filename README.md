# PHSO Complaint Forecasting

This project contains a machine learning model for forecasting complaints using the PHSO (Parliamentary and Health Service Ombudsman) dataset.

## Setup Instructions

### Prerequisites
- Python 3.x
- pip package manager
- Jupyter Notebook

### Installation

1. Install the required dependencies:
```bash
pip install -r requirements.txt
```

### Running the Project

To launch and run the complaint forecasting notebook:
```bash
jupyter notebook complaints_forecasting.ipynb
```

This will open the Jupyter notebook in your default web browser, where you can run the analysis and training cells.

## Project Structure

- `complaints_forecasting.ipynb` - Main notebook containing the forecasting model and analysis
- `requirements.txt` - Python package dependencies

## Usage

Run through the notebook cells sequentially to:
1. Load and explore the complaint data
2. Preprocess and prepare the data
3. Train the forecasting model
4. Evaluate results and generate predictions
